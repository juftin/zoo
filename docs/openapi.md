# OpenAPI Docs

## Interactive Docs

Interactive Swagger docs are available via `/docs` endpoint of the
ASGI application.

```shel
hatch run app:serve
```

[Open Interactive Docs (http://localhost:8000/docs)](http://localhost:8000/docs){ .md-button .md-button--primary }
